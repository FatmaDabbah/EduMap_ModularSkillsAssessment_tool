# ModularSkillsAssessmentTool-Team11-NEW
This modular platform provides instructors and students with a clear overview of progress in the five CASEL social-emotional learning competencies. Instructors can view class dashboards, progress graphs, student strengths, areas for improvement, alerts, and personalized intervention recommendations.

Students receive AI-generated feedback after each simulation attempt and can track their SEL development over time. The system automatically creates simulations, analyzes responses, identifies strengths and weaknesses, and generates personalized recommendations.

An integrated AI chat assistant allows instructors to ask data-based questions about classes and students, while students receive personal explanations, feedback, and learning guidance.
